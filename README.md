# TCP 통신 연습하기

## 개요

C\#에서의 TCP 통신 방법을 연습하기 위한 프로젝트

## 각 파일에서 테스트 한 것들

### 1차 - 단순 통신

TCP 통신이 어떻게 이뤄지는지에 대해 공부함
단순히 `연결 -> 메세지 송신 및 수신 -> 연결 종료`라는 하나의 프로세스만을 수행함

- 사용한 파일
  - [[Client/Sample.cs]]
  - [[Server/Sample.cs]]

### 2차 - 연결 유지

연결을 유지한 상태로 여러 메세지를 주고받고, 서로 연결을 종료시키는 방법에 대해 공부함
따라서 [[Server/Sample1.cs]]에서 연결을 유지한 상태로 최대 메세지 전송 횟수를 넘거나 연결이 끊길 때까지 메세지를 주고 받도록 변경했고,
[[Client/Sample1.cs]] 또한 동일하게 여러 메세지를 주고받을 수 있도록 함
하지만 클라이언트 측은 메세지의 수신과 송신을 따로 분리해야 원할한 통신이 가능하다는 것을 깨닫고 [[Client/Sample2.cs]] 제작을 시작함
위 파일에서 송신과 수신을 다른 메서드로 만들어 Task.WhenAny로 둘이 다른 타임라인에서 돌도록 분리했고,
두 Task가 연결 상태를 공유하기 위해 CancellationTokenSource에 대해 공부함

- 사용한 파일:
  - [[Client/Sample1.cs]]
  - [[Client/Sample2.cs]]
  - [[Server/Sample1.cs]]

### 3차 - 다른 클라이언트와 대화 (진행 중)

이제 여러 클라이언트가 메세지를 주고받게 할 것
일단 두 클라이언트끼리 채팅을 칠 수 있도록 만들게 [[Client/Sample2.cs]]는 그대로 사용하고 클라이언트를 여러개 사용할 수 있는 서버 측을 만들어 볼 예정
그 후엔 n개의 클라이언트가 동시에 메세지를 주고 받게 만들어보려고 함

- 사용한 파일:
  - [[Client/Sample2.cs]]
  - [[Server/Sample2.cs]]

### n차 - 룸 기믹 추가

한 공간에서만 메세지만 주고받는게 아니라 여러 공간을 만들어두면 사용자가 찾아내서 들어오는 기믹을 추가하려 함
들어오는게 되면 사용자가 직접 방을 생성할 수 있게도 만들어볼까?
